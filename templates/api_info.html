Standalone legal guidance API for Indian cyber crime law. No external dependencies required.

API Endpoints

POST /api/legal-advice

Main endpoint for React frontend. Send user queries and get structured legal guidance.

Request: {"message": "my account was hacked", "history": [], "language": "English"}

Response: {"type": "GUIDE", "response": "...", "crime_type": "...", "sections": [...]}

GET /api/health

Health check endpoint.

Response: {"status": "healthy", "service": "AI Justice Bot API"}

GET /api/crimes

List available crime types and keywords.

Response: {"crimes": [{"id": "hacking", "name": "Unauthorized Access", "keywords": [...]}]}

POST /chat

Legacy endpoint for backward compatibility.

Request: {"message": "user query"}

Response: {"response": "guidance text"}

Supported Crime Types

  • Online Financial Fraud - IT Act Section 66D, IPC Section 420
  • Identity Theft - IT Act Section 66C, IPC Section 419
  • Unauthorized Access/Hacking - IT Act Section 66, Section 43
  • Cyberbullying/Harassment - IT Act Section 67, IPC Section 507
  • Data Theft - IT Act Section 72, Section 43A

Usage from React

const response = await fetch('/api/legal-advice', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    message: "Someone hacked my email account",
    history: [],
    language: "English"
  })
});

const data = await response.json();
console.log(data.response); // Structured legal guidance
